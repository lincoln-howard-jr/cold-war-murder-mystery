# Cold War Murder Mystery Party

I am hosting a cold war murder mystery party. The project strcuture is as follows:

- /
  - **Character Assignments.txt** a list of all players and their assigned character
  - **scripts**: includes useful scripts that modify/create/export scripts
  - **character briefs**:
    - **source**: plain text source materials for every character brief
    - **exports**: all character briefs as exported to Pages files for end user consumption

Character briefs should contain information relevant to that character, it is expected that characters will not know all the details of the night.
When brainstorming characters, consider themes from other existing character briefs so we can pull together a cohesive story.
If brainstorming, keep concepts for the character brief in an appropriately named plain text file, these might be edited by a collaborator between requests, so this file should be the source of truth.

## Workflow

Whenever a character source is modified, the Pages and PDF exports should be regenerated and the Assignments file updated accordingly.
